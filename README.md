# Comic-Con Parking System ERD

## Overview

This project contains the **ER Diagram** for a multi-zone parking system designed for a large-scale event like Comic-Con.

It supports:

- Multiple vehicle types (Bike, Car, SUV, EV)
- Multi-level parking zones
- Reserved parking (VIP, Staff, Exhibitors, Cosplayers, EV)
- Entry & exit tracking
- Parking sessions & tickets
- Payment management

---

## Key Features

- One vehicle → multiple parking sessions
- One parking spot → reused over time
- Tracks real-time parked vehicles (`active` sessions)
- Supports reserved parking categories
- Flexible pricing based on vehicle & spot type
